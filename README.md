## Описание проекта

Адаптировано под мобильную и декстоп версию. Присутствует валидация. Реализован статус-бар. Отправка на сервер.

## Технологии, использованные в проекте

* styled-components
* Typescript
* React
* Redux-Toolkit
* RTK Query
* react-hook-form
* Yup
* imask-react
